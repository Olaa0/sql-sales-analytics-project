# SQL Sales Analytics Project

This project demonstrates a simple sales analytics data model
using fact and dimension tables.

## Project Structure
- Schema.sql – creates the database tables
- Data.sql – inserts sample data
- Analysis.sql – contains analytical SQL queries

## Skills Demonstrated
- Star schema design (fact & dimension tables)
- INNER and LEFT JOINs
- GROUP BY and HAVING
- Handling NULLs with COALESCE
- CTEs (WITH clauses)
- CASE statements
- Window functions (RANK, running totals)

## How to Run
1. Run Schema.sql to create tables
2. Run Data.sql to insert sample data
3. Run Analysis.sql to perform analysis queries
